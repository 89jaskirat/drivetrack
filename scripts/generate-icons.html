DriveTrack — App Icon Generator

Click each "Download" button to save the PNG. Then move the files to assets/ in the project, replacing the existing files.

After downloading, copy files to:
icon.png → assets/icon.png
adaptive-icon.png → assets/adaptive-icon.png
splash-icon.png → assets/splash-icon.png
favicon.png → assets/favicon.png

Then commit and trigger a new EAS build — icon changes are not OTA-updatable.
